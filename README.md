## ✨ Основные возможности

* **🔍 Автоматический анализ аудиодорожки (YouTube)**: Расширение считывает субтитры видеоролика в реальном времени и отправляет текст речи ИИ-модели для поиска и верификации цитат.
* **📖 Сверка с достоверными источниками**: Каждая найденная цитата проверяется по науке хадисов (Мусталах аль-Хадис) со ссылками на авторитетные базы (sunnah.com, quran.com).
* **👤 Оценка личности проповедника (Manhaj & Aqeeda)**:
  * Программный и ИИ-анализ личности лектора на соответствие акиде Ахлю Сунна валь-Джама'а.
  * Разделение на категории: **Саляфит** (Ahlus Sunnah), **Ихван** (Ихван аль-Муслимин), **Хариджит** (Такфириты), **Заблудший** или **Сомнительная личность** для предотвращения путаницы в призыве.
* **🌐 Поддержка любых ИИ-провайдеров**:
  * **Google Gemini API** (бесплатный ключ).
  * **Локальный ИИ (Ollama / LM Studio)** — для 100% конфиденциального и бесплатного анализа на вашем ПК.
  * **DeepSeek API / OpenRouter** — для быстрых и дешевых проверок.
* **🌍 Полная мультиязычность**: Интерфейс и подробные отчеты ИИ доступны на **Русском**, **Английском** и **Арабском** языках.

## 🛠️ Установка расширения на ПК

Поскольку расширение разработано для индивидуального использования, его можно легко установить в режиме разработчика:

1. **Скачайте исходный код** репозитория (или распакуйте архив).
2. Откройте ваш браузер (Chrome, Edge, Яндекс) и перейдите на страницу расширений:
   * В Chrome: `chrome://extensions/`
   * В Edge: `edge://extensions/`
3. В правом верхнем углу включите тумблер **«Режим разработчика»** (Developer mode).
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
5. Выберите папку проекта (в которой находится файл `manifest.json`).

## ⚙️ Настройка провайдера ИИ

После установки расширения откройте его настройки (иконка шестеренки):

### 1. Использование Google Gemini API (Бесплатно)
* Выберите провайдера **Google Gemini API**.
* Получите бесплатный ключ на сайте [Google AI Studio](https://aistudio.google.com/).
* Вставьте ключ в настройки и сохраните.

### 2. Использование OpenRouter (Бесплатные модели)
* Выберите провайдера **Локальный ИИ / Сторонний API**.
* Укажите адрес API: `https://openrouter.ai/api/v1`
* Вставьте ваш токен от OpenRouter.
* Укажите модель: `google/gemma-2-9b-it:free` или `meta-llama/llama-3-8b-instruct:free`.

### 3. Использование локального Ollama (Полностью офлайн)
* Установите [Ollama](https://ollama.com/) на ПК.
* Скачайте модель через консоль: `ollama run qwen2.5:7b`.
* В настройках укажите адрес: `http://localhost:11434/v1`, оставьте ключ пустым, укажите модель `qwen2.5:7b`.
